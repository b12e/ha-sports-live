# ha-sports-live

A Home Assistant add-on that syncs your smart lights with live football matches — goals, cards, leading-team colors, kickoff / halftime / fulltime transitions — without needing a Philips Hue Bridge.

Targets the **2026 FIFA World Cup** (June 11 – July 19, 2026), but the data layer is generic: any competition or team supported by the data provider works.

## Add as a custom add-on repository

In Home Assistant: **Settings → Add-ons → Add-on Store → ⋮ → Repositories**, add:

```
https://github.com/b12e/ha-sports-live
```

Then install the **Sports Live** add-on from the list.
